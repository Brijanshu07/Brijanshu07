## 🚀 Featured Projects

### 🗄️ MiniDB – Database Engine
> *C++ · Red-Black Tree · File I/O · CLI*

A lightweight **DBMS built from scratch** in C++ supporting `INSERT`, `SELECT`, and `DELETE` with file-based persistence across sessions.

- 🌳 Red-Black Tree indexing for **O(log n)** lookup time
- 🏗️ Clean 3-layer architecture: **parser → index → storage**
- 💪 Stress-tested with **10,000+ sequential insert/delete** operations
- 📝 1,500+ lines of modular C++ with custom memory management

---

### 🚌 DTC Bus Route Navigator
> *React.js · LeafletJS · Firebase*

An interactive map application covering **200+ live DTC bus routes** and **1,000+ stops** across Delhi.

- 🗺️ Dynamic stop markers, route highlighting, and real-time pan-and-zoom
- 🔐 Firebase authentication and session persistence
- 📱 Fully responsive on desktop and mobile
- 👥 Tested with 50+ users during development

---

### 🏥 Medical Insurance Premium Predictor
> *Python · Scikit-learn · Streamlit · Pandas*

An end-to-end ML pipeline trained on 1,300+ records achieving **~88% prediction accuracy**.

- 📊 Benchmarked Linear Regression, Random Forest, and Gradient Boosting
- ⚙️ Feature scaling, one-hot encoding, cross-validation, and hyperparameter tuning
- 🌐 Deployed via **Streamlit** for real-time browser-based predictions
